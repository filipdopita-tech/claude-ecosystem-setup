# Custom Slash Commands

| Command | Purpose | Sample invocation |
|---|---|---|
| `/render` | Orchestrate HyperFrames lint → preview → render → QC pipeline | `/render hero-intro` |
| `/storyboard` | Generate a 6-scene storyboard table via video-director subagent | `/storyboard product launch video for SaaS tool` |
| `/audit-page` | Dual-track (copy + technical) landing page audit with prioritized fix list | `/audit-page https://example.com` |
| `/brief` | Interactive project brief elicitation via brief-author agent, saved to briefs/ | `/brief website-redesign-2026` |
| `/ship` | Pre-deploy gate: leak scan + ship-checker agent, outputs GO / NO-GO | `/ship` |
| `/retro` | Session retrospective: expensive turn analysis, lessons, proposed memory entries | `/retro 2026-04-26` |
